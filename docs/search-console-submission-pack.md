## Checklist de soumission

1. Ouvrir Search Console.
2. Ajouter la propriete URL-prefix du site.
3. Utiliser la verification proposee par Google.
4. Ouvrir la section `Sitemaps`.
5. Soumettre le `sitemap.xml` correspondant.
6. Refaire exactement la meme sequence pour le second site.
